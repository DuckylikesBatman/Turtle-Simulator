Input
Language grammar (EBNF) 
program    → statement* EOF
statement  → IDENT '=' expr ';'
           | 'print' '(' expr ')' ';'
           | 'if' '(' expr ')' '{' statement* '}'
expr       → addExpr ( ('gt'|'lt'|'eq') addExpr )?
addExpr    → term   ( ('add'|'sub')         term   )*
term       → power  ( ('mult'|'div'|'mod')  power  )*
power      → factor ( 'pow' factor )?
factor     → NUMBER | IDENT | '(' expr ')' | '-' factor
           | ('sqrt'|'abs'|'floor'|'ceil') '(' expr ')'

Arithmetic : add  sub  mult  div  mod  pow
Comparisons: gt   lt   eq          (return 1=true, 0=false)
Built-ins  : sqrt  abs  floor  ceil
Control    : if
Comments   : # text to end of line

Database schema 
TABLE programs
  id        INTEGER  PRIMARY KEY AUTOINCREMENT
  name      TEXT     UNIQUE
  code      TEXT
  saved_at  TEXT     datetime('now')

TABLE runs
  id         INTEGER  PRIMARY KEY AUTOINCREMENT
  code       TEXT
  var_count  INTEGER  number of variables assigned
  had_errors INTEGER  1 = had errors, 0 = clean run
  ran_at     TEXT     datetime('now')
