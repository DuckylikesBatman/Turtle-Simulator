Mathematical Expression Interpreter

1 Lexical Analysis → 2 Syntactic Analysis → 3 Evaluation

Input
Language grammar (EBNF) 
program   → statement* EOF
statement → IDENT '=' expr ';'
expr      → term   ( ('add' | 'sub') term   )*
term      → factor ( ('mult'| 'div') factor )*
factor    → NUMBER | IDENT | '(' expr ')' | '-' factor

Ctrl+Enter to run  |  Keyword operators: add sub mult div  |  Each statement ends with ;

1 Lexical Analysis — token stream
Click "Tokens", "Run", or use Step mode
2 Syntactic Analysis — abstract syntax tree 
Click "AST", "Run", or use Step mode
3 Evaluation Trace — tree-walking execution 
Click "Run" or use Step mode to see trace
Result
Symbol Table — variable store
No variables
